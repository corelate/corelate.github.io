What is the CORELATE Platform?

Are you do you and you? The CORELATE Platform offers growing organizations flexibility to accommodate changing business processes and increasing volumes of data. Its microservices architecture allows current systems to scale independently and decoupled, making complex applications robust to change. The platform can be readily deployable on premise or on cloud, can run on any processor architecture, and can easily integrate with any data transfer protocols.

Microservice

Allows software applications flexibility by having independently deployable, customizable, and scalable components that work as a cohesive unit to serve the business objective.

Cloud-Ready

Readily deployable on premise or on cloud and is designed to take advantage of the capabilities provided by the cloud PaaS (Platform as a Service) layer.

Platform Agnostic

Runs on any combination of operating system and underlying processor architecture such as MS Windows, Mac OS, and Linux.

Database Agnostic

Supports any database management system. Tested with MySQL, Postgres, Oracle and MariaDB.

Seamless Integration

Easily integrates with data transfer protocols such as REST, SOAP, ESB, Messaging Services.

Security

The Correlate platform provide a security model that protects data integrity and privacy, and supports efficient data access and collaboration.

CORELATE Platform Features

You gain access to solutions for flexible deployment of a modern data architecture. This includes powerful tools for advanced analytics and data science, in-memory, transactional database for custom scale-out applications and more:

Supports Modern Data Architectures

Corelate Big Data Suite provides a broad foundation for modern data architectures. It can be deployed on-premise and in public clouds, and contains all the elements for batch and streaming analytics architectures.


Supports Open Standards

Corelate Big Data Suite portfolio is compatible with distributions of ODPi versions of Hadoop. All components are distributions of open source projects or are in the process of becoming open source projects.


Supports Your Data Architecture Evolution

Corelate Big Data Suite is a one to three year subscription to Corelates portfolio of big data products. It is core-based and you choose any size capacity you wish to deploy. Cores can be allocated among products as you wish for the duration of the subscription.
